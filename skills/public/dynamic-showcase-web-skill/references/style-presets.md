# Style Presets

## neo-luxury

- Tone: premium, calm, confident.
- Palette: deep black + muted gold + neutral gray.
- Motion: smooth ease-out, slightly longer transitions.
- Best for: enterprise launch, executive narrative.

## cyber-pulse

- Tone: energetic, futuristic, sharp.
- Palette: dark canvas + neon accent.
- Motion: fast, punchy, high contrast transitions.
- Best for: AI product launch, hackathon demo.

## minimal-editorial

- Tone: clear, intelligent, modern publication style.
- Palette: white/near-white base + restrained accents.
- Motion: subtle fade and slide only.
- Best for: course content, portfolio storytelling.

## brutalist-bold

- Tone: direct, loud, disruptive.
- Palette: hard contrast blocks and bold type.
- Motion: minimal transition, maximal layout impact.
- Best for: campaign pages, statement-driven presentations.

## Preset Selection Guide

- If audience is executives, start with `neo-luxury`.
- If goal is novelty and speed perception, use `cyber-pulse`.
- If clarity and reading comfort matter most, use `minimal-editorial`.
- If differentiation is the top priority, use `brutalist-bold`.
